About WebServ

WebServ is a high-performance HTTP/1.1 server implementation written in C++98. It was created as part of the 42 school curriculum to understand the fundamentals of web servers and network programming.

This server implements core HTTP features including:

  • Non-blocking I/O using poll()
  • Multiple server instances on different ports
  • GET, POST, and DELETE methods
  • Static file serving
  • File upload functionality
  • CGI support for dynamic content
  • Custom error pages
  • Directory listing
